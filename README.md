### Explainable Quantum-Inspired Multi-label NFR Classification

[![Repo](https://img.shields.io/badge/Repository-Quantum%20NFR%20Research-181717?style=flat-square&logo=github)](https://github.com/Lancelot-sys25/Quantum-Inspired-NFR-Classification)

A scientific research prototype for classifying non-functional requirements with explainable machine learning.

- Implemented quantum-inspired semantic projection and hybrid Quantum-SVM experiments.
- Compared against TF-IDF Logistic Regression and Linear SVM baselines.
- Reported Micro-F1, Macro-F1, Hamming Loss, LRAP, cross-validation, ablation study, and token-level explanation outputs.

**Tech stack:** Python, scikit-learn, NumPy, pandas, LaTeX, NLP/Machine Learning
